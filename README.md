How to Run my program:
Must have Java JDK installed and a functioning IDE or terminal to use. Files must be compiled with javac command. Davis_Final.java must be compiled. Printed prompts will require keyboard input into the terminal.

Short summary/features:
This program is a personality test that uses four classes to run. When running the program, a menu with the options to take the test, reset the score and restart the test, check the score, and show personality descriptions will be printed. The test has 16 questions which are scored corresponding the the user response. Answer choices 1-4 in the test will be scored, while answer choice 5 will reprint the menu. Based on the score, a personality description will be printed after all questions are taken.

Specific inputs, expected outputs, variables:
All inputs will be made by the keyboard. In the menu, numbers 1-4 will be input to: 1. Take Test, 2. Restart Test, 3. View Score, and 4. View Descriptions. When 1 is chosen the questions will be printed sequentially, with 5 options presented. 
Example input and output: 

1) Take Test
2) Restart Test
3) View Score
4) View Descriptions
Choose 1-4: 1 //Input

//Output

People may describe you as: 
1. Enthusiastic and optimistic.
2. Analytical and private.
3. Driven and motivated.
4. Laid-back and accommodating.
5. Return to Menu
